Refresh access token

Exchange a valid refresh token for new access token. The refresh token is read from HTTP-only cookie (not request body). A new refresh token cookie is set on success (token rotation).

cookie Parameters
refresh_token
required
string

HTTP-only cookie containing refresh token

Responses

Development server

http://localhost:3000/auth/refresh

Response samples

  • 200
  • 401
Content type
application/json

  • "accessToken": "string",
  • "expiresIn": 900,

bearerAuth
